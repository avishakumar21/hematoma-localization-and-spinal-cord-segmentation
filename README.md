This repo contains all the code used for the preprocessing of the spinal cord Ultrasound B-Mode Sagittal images. 

The directory DataScaling contains the codes required to scale all the images. 

The directories beginning with HematomaDetection contains the code used to train object detection models for localizing the hematoma. The directories beginning with Segmentation contain the code used to train the sematic segmentation models for segmenting soft tissue in ultrasound spinal cord images.
